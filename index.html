✨ Key Features

🪙

Real-Time Token Counter

Uses a local tiktoken o200k_base tokenizer (native to Claude 3 & 3.5) to provide accurate offline token estimations of your active prompt context.

⏱️

Prompt Cache Countdown

Anthropic caches active prompts for up to 5 minutes. Live countdown shows when cache expires so you can query before it hits 0:00 to save tokens.

📊

Rolling Limits Tracker

Hooks directly into Server-Sent Events to capture session parameters, visualizing your 5-hour and 7-day rolling usage percentages.

📦 Simple Setup Guide

  • 1
    Download & Extract:

    Download the source ZIP and extract it to a folder on your computer.

  • 2
    Enable Developer Mode:

    Open your browser and navigate to chrome://extensions/, then toggle the Developer mode switch in the top-right corner.

  • 3
    Load Extension:

    Click the Load unpacked button on the top-left, and select the folder you extracted in Step 1.

Enable the token counter inside the official **Claude Desktop Application** (macOS and Windows) using automated scripts:

  • 1
    Download Extension:

    Get the project ZIP and unzip it.

  • 2
    Run Patch Script:

    Ensure you have Node.js installed, then run the patch script matching your OS:

     macOS: Double-click patch-mac.command.

    🪟 Windows: Double-click patch-windows.bat.

    Security Prompt: If macOS blocks the script execution, open your Terminal, navigate to the folder, and run:
    node scripts/patch-desktop.js
  • 3
    Relaunch Application:

    Completely exit and restart the official **Claude Desktop** application to see the injected token bar active!
